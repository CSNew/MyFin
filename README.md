### ขั้นตอนที่ 3: แก้ไขไฟล์ `index.html` และ `App.jsx`

แอปที่คุณสร้างไว้เป็น React Component หลัก (`App.jsx`) ซึ่งต้องถูกเรียกใช้ผ่านไฟล์ `index.html` ครับ

#### A. อัปเดตไฟล์ `public/index.html`

โปรดสร้างหรือแก้ไขไฟล์ที่อยู่ในโฟลเดอร์ **`public/`** โดยมีเนื้อหาดังนี้:


http://googleusercontent.com/immersive_entry_chip/1

#### B. แก้ไข `src/App.jsx` (เล็กน้อย)

ไฟล์ที่คุณมีอยู่เป็นแค่ Component หลัก แต่ในการรัน React ในโปรเจกต์จริง คุณต้องมีไฟล์ **`index.js`** เพื่อเชื่อมต่อ Component หลักเข้ากับ DOM ครับ

เราจะเปลี่ยนชื่อไฟล์ `App.jsx` ให้เป็น `App.js` หรือคงไว้ก็ได้ แต่เราต้องสร้างไฟล์ **`src/index.js`** ขึ้นมา:
